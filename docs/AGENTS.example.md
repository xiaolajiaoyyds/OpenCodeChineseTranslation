## 危险操作确认

### 必须确认

- Git: `push --force`、`reset --hard`、`rebase`
- 文件: `rm -rf`、批量删除 5+ 文件
- 配置: Package.swift、.env、Entitlements

### 确认格式

```

操作: [具体命令]

```
